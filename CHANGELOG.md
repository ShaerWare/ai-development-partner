# Changelog

All notable changes to AI Development Partner will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Структура шаблонов для различных платформ (Claude, ChatGPT, локальные LLM, VS Code)
- Подробное руководство по интеграции (INTEGRATION_GUIDE.md)
- Примеры использования для Laravel проектов
- CONTRIBUTING.md с правилами контрибуции
- Поддержка VS Code через расширение Continue

### Changed
- Улучшена структура репозитория (templates/, examples/, prompts/)
- Обновлена документация с учетом новой структуры

## [4.1.0] - 2026-01-05

### Added
- Универсальный системный промт-шаблон для создания любого ИИ-ассистента
- Поддержка различных областей применения (не только разработка)

### Changed
- Улучшена структура команд
- Оптимизирован размер промпта для лучшей совместимости с бесплатными планами

## [4.0.0] - 2025-10-03

### Added
- Система быстрых команд с префиксом `/`
- Команда `/tz` для формирования технического задания
- Команда `/sprints` для разбивки задач на спринты
- Команда `/pr` для создания Pull Request workflow
- Команда `/prompt` для вывода промпта целиком
- Команда `/nocode` для стратегического анализа без генерации кода
- Команда `/audit` для критического аудита решений
- Команда `/stop` для перечитки контекста
- Команда `/help` для справки по командам
- Роль Критического Аудитора для сложных задач (сложность 7+)
- PM-модуль с оценкой стоимости и ROI
- Поддержка проактивного анализа неочевидных аспектов (для задач 5+)

### Changed
- Улучшен workflow с обязательным запросом контекста при модификации кода
- Расширены роли: добавлены PM, DevOps, Бизнес-консультант
- Улучшена структура формата ответов
- Оптимизирована генерация CRUD с автоматическими тестами и Swagger

### Removed
- Неиспользуемые команды из предыдущих версий

## [2.0.0] - 2025-09-29

### Added
- Поддержка бизнес-анализа и ROI расчетов
- Проактивный анализ безопасности (SQL injection, XSS, CSRF)
- Рекомендации по оптимизации производительности
- Автоматическая генерация Feature-тестов для Laravel
- Полная Swagger документация для API endpoints

### Changed
- Улучшен контекст проекта с финансовыми параметрами
- Расширены роли ассистента (Системный Аналитик, Tech Lead, Тимлид)

## [1.0.0] - 2025-09-01

### Added
- Базовый системный промпт для разработки
- Поддержка Laravel CRUD генерации
- Автоматическое создание моделей, миграций, контроллеров
- Генерация API Resources
- Базовая валидация через FormRequests
- Роуты для API
- Базовые принципы работы (SOLID, DRY, KISS, YAGNI)

---

## Типы изменений

- `Added` - новая функциональность
- `Changed` - изменения в существующей функциональности
- `Deprecated` - функции, которые скоро будут удалены
- `Removed` - удаленные функции
- `Fixed` - исправления багов
- `Security` - исправления уязвимостей
