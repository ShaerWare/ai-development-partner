# AI Development Partner для ChatGPT

## Установка в ChatGPT:

1. Откройте [ChatGPT](https://chat.openai.com)
2. Нажмите на ваше имя (левый нижний угол)
3. Выберите `Settings` → `Personalization` → `Custom Instructions`
4. Включите опцию `Enable for new chats`

## Заполнение Custom Instructions:

ChatGPT имеет два поля для custom instructions:

### Поле 1: "What would you like ChatGPT to know about you to provide better responses?"

Вставьте информацию о вашем проекте:

```markdown
## О моём проекте:
- Технический стек: PHP Laravel, MySQL, Apache
- Стратегическая цель: Создать SaaS продукт с выручкой 500к₽/мес
- Команда: 1 разработчик + AI ассистент
- Бюджет: 50000₽/неделя на разработку

## Мои роли:
Я - технический предприниматель, разрабатываю веб-приложения.

## Что мне важно:
- Качественный, масштабируемый код
- Оценка стоимости и ROI решений
- Безопасность и производительность
- Понятная документация
```

### Поле 2: "How would you like ChatGPT to respond?"

Вставьте содержимое промпта из `/prompts/system_prompt_full.md` или `/prompts/system_prompt_mini.md` (в зависимости от вашего плана).

**Важно:** ChatGPT имеет ограничение на размер Custom Instructions. Если полная версия не помещается, используйте мини версию (`system_prompt_mini.md`).

## Особенности работы с ChatGPT:

1. **Память контекста:** ChatGPT запоминает информацию между сессиями через Memory feature
2. **Длина ответов:** Может потребоваться попросить "продолжи" для длинных ответов
3. **Code Interpreter:** Доступен в ChatGPT Plus для анализа кода и данных

## Альтернативный способ (для каждого чата):

Если Custom Instructions не работают или недоступны:

1. Создайте новый чат
2. В первом сообщении отправьте:
```
Я хочу, чтобы ты работал как мой AI Development Partner.

[Вставьте сюда содержимое промпта]

Подтверди, что ты понял инструкции и готов работать.
```

## Проверка:

```
/help
```

ChatGPT должен вывести таблицу команд.

## Рекомендации:

- Используйте GPT-4 или GPT-4 Turbo для лучших результатов
- GPT-3.5 может работать с мини версией промпта
- Включите Code Interpreter для работы с кодом
