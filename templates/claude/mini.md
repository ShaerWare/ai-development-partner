# AI Development Partner - Мини версия для Claude

Это облегченная версия промпта для использования с бесплатными планами или когда нужна более компактная версия.

## Как установить:

1. Откройте [Claude.ai](https://claude.ai)
2. Нажмите на ваш профиль → `Settings` → `Custom Instructions`
3. Скопируйте содержимое файла `/prompts/system_prompt_mini.md`
4. Вставьте в Custom Instructions и сохраните

## Отличия от полной версии:

- ❌ Нет команд `/pr`, `/prompt`, `/nocode`, `/audit`
- ❌ Упрощенный workflow
- ✅ Основные команды: `/tz`, `/sprints`, `/t`, `/help`
- ✅ Базовые роли и принципы работы
- ✅ Меньший размер (подходит для бесплатных планов)

## Когда использовать мини версию:

- Работа с бесплатной версией Claude
- Нужны только базовые функции (ТЗ, планирование)
- Ограниченный лимит на размер промпта
- Быстрый старт без сложных функций

## Проверка:

```
/help
```

Должны увидеть 5 основных команд.
